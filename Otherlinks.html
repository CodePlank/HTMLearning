Other types of links

  • You can use mailto links with this format after a basic anchor tag : "mailto:example@truco.com" (you can even add "?subject" to prefill it)mailto example
  • Links in a new window, the target tag followed by _blank allows that : Alderiate in new window
  • Link to a specific part of same page :
    You first need to give an ID attribute to any element, example : < h1 id="top" >
    then use the "#" symbol in an anchor tag followed by the id attribute of the element required
