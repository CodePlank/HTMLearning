Introduction to images

If you are creating a website from scratch, it is good practice to create a folder for all the images the website will use.

Adding Images

To add an image into the page :

  • use the tag < img >. This is an empty element, meaning it doesn't have a closing tag.
  • src tells the browser where it can find the image file (usually a relative URL)
  • alt provides a description of the image if you can't see it (use by people with visual impairments or search engines, for instance)
  • title can also be used with the img element to provide additional info (usuallydisplayed in a tooltipwhen the image is hovered)
  • Height and Width are used to specify the size of the image

Figure and figcaption tags

The figure element is made to associate an image and it's caption. Several images can be in the same figure as long as they share the same caption
