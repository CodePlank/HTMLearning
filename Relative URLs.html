Relative URLS, How does it work?

Relative URLs can be used when linking pages within your own website. They are a shorthand way of telling the browser where to find your files

Relative link types

  • Same folder : Just use the name file, nothing else is needed
  • Child folder : name of the child folder, forward slash, then file name
  • Grandchild folder : same process
  • Parent folder : use to indicate the folder above the current one
  • Grandparent folder : same process
