Tables

BASIC TABLE STRUCTURE

The table element < table > is use to to create a table, the content and rows are written inside this tag.
The < tr > tag (table row) indicates the start of each row. it is followed by one or several < td > elements (one for each cell in that row)
Each cell of a table is represented using < td >,

Example

THIS is a table
see? it's easy
to make one

TABLE HEADINGS

< th > element is used like the < td > but it represents the heading of a row/column (th stands for table heading)
Even if a cell has no content, you still need to use a td or th element to represent the presence of an empty cell or it will mess up the table rendering
You can use the scope element to indicate whether it's a column or row. To do so, use the value "row" or "col" to indicate them respectively

Example

Streamer ElTruco Gross Gore
Player Jankos Fakeru

Spanning Columns

Sometimes you need the entries in a table to stretch access more than one column
The < colspan >can be attribute, used in a td or th element to indicate how many columns this cell should run accross.

The same thing might be used for rows, < rowspan > can be used equally on th and td elements

TABLE HEADER, BODY AND FOOTER

3 elements can help distinguish between the main content of a table, the first and last row

  • < thead > The headings of the table should be inside this element
  • < tbody > The body of the table should be inside this element
  • < tfooter > The footer of the table should be inside this element

NOTES ON OLD CODE: WIDTH, SPACING, BORDER AND BACKGROUND.

These have been replaced by CSS, but you might come accross them sometimes.
You can use the following attributes :

  • The width attribute could be used in the table tag or any td/th element to indicate how wide this element should be.
    The columns in a table need to form a straight line, so you will very often see this attribute on the first row, the following rows will adapt.
  • The opening table tag could also use the cellpadding attribute to create space between each cell
  • The border attribute was used to add... well a border. Followed by "=x" x being the width of the border in pixel.
  • the bgcolor attribute was used to set the background color, and could be used for the whole table or individual cells.
